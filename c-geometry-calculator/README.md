# Geometry Calculator in C

Console application written in C for calculating area and perimeter of basic geometric shapes.

## Features

- Square (area and perimeter)
- Rectangle (area and perimeter)
- Circle (area and perimeter)
- Menu-based navigation
- Input validation

## Technologies

- C programming language
- Standard libary (stdio.h)
